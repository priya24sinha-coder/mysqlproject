# mysqlproject
Post Office Parcel Tracking System


This project simulates a real-world parcel tracking system for a post office.
You can:
✔ Track parcel movement
✔ Analyze delivery delays
✔ Identify high-performing branches
✔ Calculate delivery time performance
✔ Generate business insights



How to Run------->

mysql -u root -p < schema.sql
mysql -u root -p < sample_data.sql


Step 1 — Open Terminal / CMD
Step 2 — Login into MySQL


Run schema.sql
mysql -u root -p < schema.sql

Run sample_data.sql
mysql -u root -p < sample_data.sql

Run analysis_queries.sql
mysql -u root -p post_office < analysis_queries.sql
